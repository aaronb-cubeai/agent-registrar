# Audit Log

This folder contains weekly performance reviews written by the Auditor Agent.

## File Naming Convention

```
audits/YYYY-MM-DD-<agent-name>.md
```

Example: `audits/2026-03-23-aaron-sales-agent.md`

## Review Format

Each audit file contains:
- **Performance summary** — what the agent did well
- **Issues identified** — failures, drift, or gaps
- **Improvement suggestions** — specific actionable changes
- **Follow-up status** — did the agent act on last week's suggestions?

---
*Audits are written by the Auditor Agent and are read-only for other agents.*
